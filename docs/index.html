OVERVIEW

The AI Black Box, opened.

Strata is an open-source desktop tool for AI model profiling and visualization. Load any PyTorch or ONNX model, run inference on real inputs, and explore every layer's full tensor output in an interactive 3D graph.

FEATURES

01

3D ARCHITECTURE GRAPH

Navigate your model's full layer structure in an interactive 3D graph. Pan, rotate, and zoom through every connection.

02

REAL-TIME INFERENCE

Watch activations propagate through your network layer by layer as inference runs. Every node lights up with live tensor data.

03

FULL TENSOR EXPORT

Export complete, untruncated tensor values to .txt for any layer. Built for engineers who need the actual numbers.

04

FEATURE MAP PREVIEW

Click any layer after inference to see its output rendered as a grayscale feature map. Watch raw pixels become semantic features.

05

PYTORCH + ONNX

Native support for PyTorch nn.Module models and ONNX exports. Works with vision models, transformers, and custom architectures.

06

NO CLOUD

Entirely local. Your models and data never leave your machine. No account required.
